# 攻克辅音

拼音字母是带尾音的，而英语字母是不带的。

> /w/ 和 /r/ 发音时，一定要注意撅嘴。

> /n/ 发音时，舌尖一定要抵住上齿龈。

### 摩擦音 /z/ 

| **s** | **z** |
| :--- | :--- |
| close | zoo /zuː/ |
| rose | zipper /ˈzɪpɚ/ 拉链 |
| does 工作 | zebra /ˈziːbrə/ 斑马 |
| these /ðiːz/ | zero /ˈzɪroʊ/ |

> zoo 注意撅嘴。
> zipper 在美音里，p 浊化成 b。
> zebra 的英美发音略有不同。
> zero 在英音里读 /ˈzɪərəʊ/，比美音的 /ˈzɪroʊ/ 中间多了一个 /ə/。

### 鼻音 /n/

> n 的舌位和 l t d 都是一样的，从舌边发音，而且音是从鼻子发出来的。

| ten | knee |
| pen | knob /nɒb/ 把手 |

> 注意 nine 尾巴上还有个 n
